## 1. 화면 구성 개요

### 1.1 인터페이스 유형

| 유형 | 설명 | 기술 |

### 1.2 대시보드 구성

```
┌─────────────────────────────────────────────────────────────┐
│                    Streamlit 대시보드 구조                    │

│  │                    [SCR-001]                        │   │

│                           │                                 │
│      ┌────────────────────┼────────────────────┐           │
│      │                    │                    │           │
│      ▼                    ▼                    ▼           │
│  ┌────────┐          ┌────────┐          ┌────────┐       │
│  │모듈별   │          │워크플로│          │ 통계   │       │

### 3.1 전체 화면 플로우

```
┌─────────────────────────────────────────────────────────────────────┐
│                           화면 플로우                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────────┐                                                  │
│  │ 메인 대시보드 │ ◀─────────────────────────────────────┐         │
│  │  (SCR-001)   │                                       │         │
│  └──────┬───────┘                                       │         │
│         │                                               │         │
│    ┌────┴────┬────────┬────────┬────────┐              │         │
│    │         │        │        │        │              │         │
│    ▼         ▼        ▼        ▼        ▼              │         │
│ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐          │         │
│ │SCR-  │ │SCR-  │ │SCR-  │ │SCR-  │ │SCR-  │          │         │
│ │002   │ │003   │ │004   │ │005   │ │006   │          │         │
│ │뉴스  │ │RAG   │ │블로그│ │품질  │ │이미지│          │         │
