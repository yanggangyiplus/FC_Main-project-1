## 1. 서비스 소개

**Auto Blog**는 AI를 활용한 자동 블로그 생성 시스템입니다.

### 주요 기능
- 네이버 뉴스에서 최신 기사 수집
- AI가 블로그 글 자동 작성
- AI 이미지 자동 생성
- 네이버 블로그 자동 발행

### 작동 흐름
```
뉴스 수집 → 분석 → 블로그 작성 → 품질 평가 → 이미지 생성 → 발행
```

## 2. 시작하기 전 준비사항

### 2.1 필요한 API 키

이 서비스를 사용하려면 **본인의 API 키**가 필요합니다.

| 항목 | 필수 여부 | 용도 |
|------|----------|------|
| Google API Key | ✅ 필수 | AI 글 작성 및 이미지 생성 |
| 네이버 계정 | ✅ 필수 | 블로그 발행 |
| Slack Token | ⚪ 선택 | 완료 알림 |

### 2.2 Google API Key 발급 방법

1. [Google AI Studio](https://aistudio.google.com/app/apikey) 접속
2. Google 계정으로 로그인
3. **"Create API Key"** 버튼 클릭
4. 생성된 키 복사 (AIzaSy...로 시작)

> 💡 **무료 사용량**: 분당 15회 요청, 일일 1,500회 요청 (2024년 12월 기준)

### 2.3 네이버 블로그 준비

1. [네이버](https://naver.com) 계정이 없다면 회원가입
2. [네이버 블로그](https://blog.naver.com) 개설
3. 블로그 주소 확인: `https://blog.naver.com/본인아이디`

> ⚠️ **주의**: 네이버 2단계 인증이 켜져 있으면 자동 발행이 안 될 수 있습니다.

## 3. 사용 방법

### Step 1: 서비스 접속

운영자가 제공한 URL로 접속합니다.
(예: `https://auto-blog-xxx.streamlit.app`)

### Step 2: API 키 입력

1. 왼쪽 사이드바에서 **"🔑 API 키 설정"** 클릭
2. 각 항목 입력:
   - **Google API Key**: 위에서 발급받은 키
   - **네이버 아이디**: 블로그 발행용 네이버 ID
   - **네이버 비밀번호**: 네이버 비밀번호
3. API 키 상태가 **"✅ 설정됨"**으로 표시되는지 확인

### Step 3: 카테고리 선택

메인 화면에서 뉴스 카테고리를 선택합니다:
- **정치 (Politics)**: 정치 관련 뉴스
- **경제 (Economy)**: 경제/금융 뉴스
- **IT/기술 (IT & Technology)**: IT, 과학, 기술 뉴스

### Step 4: 워크플로우 실행

**"🚀 전체 워크플로우 실행"** 버튼을 클릭합니다.

### Step 5: 진행 상황 확인

각 단계가 순차적으로 실행됩니다:

| 단계 | 설명 | 예상 시간 |
|------|------|----------|
| 1️⃣ 뉴스 | 최신 뉴스 수집 | 30초~1분 |
| 2️⃣ RAG | 기사 분석 및 저장 | 10초 |
| 3️⃣ 블로그 | AI가 글 작성 | 30초~1분 |
| 4️⃣ 평가 | 품질 검사 (최대 3회) | 30초~2분 |
| 5️⃣ 이미지 | AI 이미지 생성 | 30초~1분 |
| 6️⃣ 인간화 | 문체 개선 | 20초 |
| 7️⃣ 발행 | 네이버 블로그 발행 | 1~2분 |

**총 예상 시간: 3~7분**

### Step 6: 결과 확인

- **성공 시**: 발행된 블로그 URL이 표시됩니다
- **실패 시**: 오류 메시지와 해결 방법이 표시됩니다

## 4. 자주 묻는 질문

### Q1: Google API Key는 무료인가요?

**A**: 네, 무료 사용량이 있습니다.
- Gemini 2.5 Flash: 분당 15회, 일일 1,500회
- 일반적인 사용에는 충분합니다

### Q2: 네이버 로그인이 안 됩니다

**A**: 다음을 확인하세요:
1. 아이디/비밀번호 정확히 입력
2. 네이버 2단계 인증 **비활성화**
3. 처음 사용 시 네이버에서 "새로운 기기 로그인" 알림 허용

### Q3: 품질 평가에서 계속 실패합니다

**A**: AI가 생성한 글이 품질 기준(80점)을 3회 연속 미달하면 발행이 중단됩니다.
- 다른 카테고리로 시도해보세요
- 시간을 두고 다시 시도해보세요 (뉴스 업데이트)

### Q4: 이미지가 생성되지 않습니다

**A**: Google API의 안전 필터에 의해 차단될 수 있습니다.
- 민감한 주제(정치적 인물, 폭력 등)는 이미지 생성이 제한됩니다
- 다른 주제로 시도해보세요

### Q5: 발행된 글은 어디서 확인하나요?

**A**: 발행 성공 시 표시되는 URL을 클릭하거나, 본인의 네이버 블로그에서 확인할 수 있습니다.

### Q6: 하루에 몇 개까지 발행할 수 있나요?

**A**: API 무료 한도 내에서 약 **10~20개** 정도 발행 가능합니다.
- Google API 요청 수에 따라 달라집니다

### Q7: 발행된 글을 수정할 수 있나요?

**A**: 네, 네이버 블로그에서 직접 수정 가능합니다.
- 네이버 블로그 관리 > 글 관리에서 수정

### Q8: API 키는 안전한가요?

**A**:
- API 키는 브라우저 세션에만 저장됩니다
- 페이지를 새로고침하면 다시 입력해야 합니다
- 서버에 저장되지 않습니다
