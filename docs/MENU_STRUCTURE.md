## 3. 메뉴 ID 체계

### 3.1 메뉴 ID 명명 규칙

```
형식: M-XXX-YY

M     : Menu의 약자
XXX   : 대분류 번호 (001, 002, 003)
YY    : 소분류 번호 (01, 02, ...)

M-001    : 메인 대시보드
M-001-01 : 메인 대시보드 > 개요 탭
```

### 5.2 사이드바 구현 예시

```python
# dashboards/common.py

import streamlit as st

def render_sidebar():
    """공통 사이드바 렌더링"""
    with st.sidebar:
        st.title("Auto Blog")
        st.caption("자동 블로그 생성 시스템")

        st.divider()

        st.subheader("네비게이션")
        if st.button("🏠 메인 대시보드", use_container_width=True):
            st.switch_page("dashboards/main_dashboard.py")

        st.divider()

        st.subheader("모듈")
        modules = [
            ("📰 뉴스 스크래퍼", "dashboard_01_news_scraper.py"),
            ("🧠 RAG 빌더", "dashboard_02_rag_builder.py"),
            ("✍️ 블로그 생성기", "dashboard_03_blog_generator.py"),
            ("⭐ 품질 평가", "dashboard_04_critic_qa.py"),
            ("🖼️ 이미지 생성기", "dashboard_05_image_generator.py"),
            ("💬 문체 개선", "dashboard_06_humanizer.py"),
            ("📤 블로그 발행", "dashboard_07_blog_publisher.py"),
            ("📢 알림", "dashboard_08_notifier.py"),
        ]

        for name, file in modules:
            if st.button(name, use_container_width=True):
                st.switch_page(f"dashboards/{file}")

        st.divider()

        st.subheader("워크플로우")
        if st.button("⚡ 전체 실행", use_container_width=True):
            st.switch_page("dashboards/workflow_dashboard.py")

        st.divider()

        st.caption("버전: 1.0 | 상태: 정상")
```
